# Tickers
Small software to show interesting yahoo finance real time last info in a terminal using tickers.

## Example Display
```
+------------------+-----------+----------+---+
| 2023-03-09 11:55 | EURUSD=X  |  1.05708 | + |
+------------------+-----------+----------+---+
| 2023-03-09 11:55 | SI=F      | 20.16    | + |
+------------------+-----------+----------+---+
| 2023-03-09 11:55 | WAVES-EUR |  2.02332 | - |
+------------------+-----------+----------+---+
```
